# ScarletFlow

小红书直播自动回复 Android App

## 功能特性

- **自动发送消息** - 在小红书直播间自动定时发送弹幕
- **多条内容随机** - 支持设置多条回复内容，每次随机选择一条发送
- **拟人输入模式** - 根据内容长度模拟真人打字速度（每字150-300ms）
- **随机间隔** - 发送间隔有±30%随机波动，避免机械式发送
- **悬浮控制按钮** - 可在任意界面控制启停，无需切换App
- **自定义间隔** - 支持自定义发送间隔时间

## 使用方法

1. **安装 App**
2. **开启无障碍服务**
   - 点击「开启无障碍服务」
   - 在系统设置中找到「ScarletFlow」并开启
3. **设置回复内容**
   - 输入要发送的内容，每行一条
   - 多条内容会随机发送
4. **设置发送间隔**
   - 输入间隔秒数（建议10秒以上）
5. **开启拟人模式**（推荐）
   - 根据内容长度自动计算打字时间
6. **开始使用**
   - 点击「显示悬浮按钮」
   - 打开小红书进入直播间
   - 点击悬浮按钮开始/停止

## 技术实现

- **语言**: Kotlin
- **最低版本**: Android 7.0 (API 24)
- **核心技术**: AccessibilityService (无障碍服务)
- **UI**: Material Design 3

## 注意事项

- 需要授予无障碍服务权限和悬浮窗权限
- 建议发送间隔不少于10秒，避免触发平台风控
- 小红书更新可能导致界面识别失败，需要适配
- 仅供学习交流使用
